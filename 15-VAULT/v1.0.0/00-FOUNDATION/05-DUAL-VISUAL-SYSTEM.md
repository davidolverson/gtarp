# Palm6 Dual Visual System

**Version:** v1.0.0  
**Status:** Approved v1.0.0 (Phase 0, DEC-004, 2026-07-18)

---

Palm6 uses **two distinct but complementary visual systems**. Understanding the difference between them is essential.

## System A - Identity (The Logo System)

**Purpose:** This is the official, timeless representation of Palm6.

**Characteristics:**
- Simple
- Minimal
- Luxury
- Timeless
- Scalable
- Works in one color (solid black on white)

**Examples of brands that follow this approach:**
- Porsche
- Rolex
- Apple
- Rimowa

**What it is used for:**
- Primary wordmark
- P6 monogram / icon
- Hero crest
- Favicon
- Official documents
- Merchandise (embroidery, engraving, etc.)

**Rule:** The Identity system must never rely on cinematic effects, glows, backgrounds, or dramatic lighting to look good.

## System B - Marketing (The Emotional System)

**Purpose:** This is where we create emotion, atmosphere, and the "GTA6 feel."

**Characteristics:**
- Cinematic
- Atmospheric
- High production value
- Rich in storytelling and world flavor
- Can be dramatic, vibrant, or moody

**What it is used for:**
- Trailers and key art
- Loading screens
- Social media assets
- Website hero sections
- Event marketing
- In-game environmental storytelling (when appropriate)

**Important Rule:**

The logo itself belongs to **System A**.  
Marketing assets can be rich and cinematic, but the core Identity (logo, wordmark, monogram) must remain clean and timeless.

This separation prevents the most common mistake in roleplay branding: trying to make the logo carry all the emotion. The logo should be strong enough to stand alone. The emotion comes from how we use it in context.
