# 15-VAULT - Approved Assets Only

The Vault holds **only** materials that have reached **Approved** status through the
Decision Log. It is the single place where "final / production-ready" lives.

## The one rule

Nothing enters this folder until it has an **Approved** entry in
`00-FOUNDATION/DECISION-LOG/`. No exceptions. Commercial / sellable assets carry extra
protection and must never bypass the Asset Lifecycle
(Experimental → Candidate → Approved → Vault).
